TanStack Form is compatible with various front-end frameworks, including React, Vue, Angular, Solid, Svelte, and Lit. Install the corresponding adapter for your framework using your preferred package manager.

## Framework Adapters

Choose the package for your framework:

<Tabs>
  <Tab title="React">
    <CodeGroup>
      ```bash npm
      npm install @tanstack/react-form
      ```
      
      ```bash pnpm
      pnpm add @tanstack/react-form
      ```

      ```bash yarn
      yarn add @tanstack/react-form
      ```

## Developer Tools

Developer tools are available using [TanStack Devtools](https://tanstack.com/devtools/latest) for React and Solid. Install the devtools packages as dev dependencies to debug forms and inspect their state.

<Tabs>
  <Tab title="React">
    <CodeGroup>
      ```bash npm
      npm install -D @tanstack/react-devtools @tanstack/react-form-devtools
      ```
      
      ```bash pnpm
      pnpm add -D @tanstack/react-devtools @tanstack/react-form-devtools
      ```
      
      ```bash yarn
      yarn add -D @tanstack/react-devtools @tanstack/react-form-devtools
      ```
      
      ```bash bun
      bun add -D @tanstack/react-devtools @tanstack/react-form-devtools
      ```
    </CodeGroup>
  </Tab>
  
  <Tab title="Solid">
    <CodeGroup>
      ```bash npm
      npm install -D @tanstack/solid-devtools @tanstack/solid-form-devtools
      ```
      
      ```bash pnpm
      pnpm add -D @tanstack/solid-devtools @tanstack/solid-form-devtools
      ```
      
      ```bash yarn
      yarn add -D @tanstack/solid-devtools @tanstack/solid-form-devtools
      ```
      
      ```bash bun
      bun add -D @tanstack/solid-devtools @tanstack/solid-form-devtools
      ```
    </CodeGroup>
  </Tab>
</Tabs>

<Note>
Depending on your environment, you might need to add polyfills. If you want to support older browsers, you need to transpile the library from `node_modules` yourself.
</Note>
